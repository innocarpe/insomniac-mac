# 전체 테스트 수행 & 이슈 수정

1. 전체 테스트 수행, 정상적으로 통과하는지 확인
2. 컴파일 오류가 발생하거나 실패하는 테스트가 있을 경우, 컴파일 오류를 가장 먼저 수정. 수정 후 `.claude/commands/commit-changes.md` 룰에 맞게 커밋
3. 실패하는 테스트가 존재할 경우, 이를 어떻게 수정하면 좋을지 ultrathink 해서 먼저 계획 후 사용자에게 수정할지 문의
4. 사용자가 하락하면 모든 테스트가 통과할 때까지 수정을 계속해서 진행
5. 모든 테스트가 통과하면 수정한 내용을 정리해 사용자에게 리포트
6. 수정 내역을 루트폴더 기준 `.claude/commands/core/commit-changes.md` 룰에 맞게 커밋
